# RealTime_ChatApp



Brief description.

## Tech

- Frontend: React
- Backend: Node
- Auth: JWT
- DB: MongoDB
- Realtime: Socket.io
- CSS: Styled Components

## Setup

1. Clone repo
2. Frontend: `cd public && npm install && npm start`
3. Backend: `cd server && npm install && npm start`

## Features

- User auth
- Realtime chat
- Responsive UI
